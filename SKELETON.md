# Travel & Tourism Website — Skeleton Plan

## Site Map (Pages)

```
1. Home
2. Explore / Destinations (listing page)
3. Destination Detail
4. Login
5. Register
6. My Trips (dashboard, logged-in users)
7. Itinerary Detail / Builder
8. Profile / Account Settings
9. 404 / Not Found
```

---

## 1. Home Page
- Header / Navbar (logo, nav links, login/profile button)
- Hero section (banner, tagline, search bar)
- Featured/highlighted destinations section
- Call-to-action section (e.g. "Plan your trip")
- Footer (links, contact, socials)

## 2. Explore / Destinations Page
- Search bar
- Filter panel/sidebar (empty for now — filters TBD)
- Destination grid/list (cards)
- Pagination controls
- Empty state (no results)

## 3. Destination Detail Page
- Image gallery/banner
- Title, location, key info block
- Description section
- Ratings/reviews section
- Review submission form (logged-in only)
- Related/nearby destinations (optional section)

## 4. Login Page
- Login form
- Link to Register

## 5. Register Page
- Registration form
- Link to Login

## 6. My Trips (Dashboard)
- List of user's saved itineraries (cards)
- "Create new trip" button
- Empty state (no trips yet)

## 7. Itinerary Detail / Builder Page
- Trip title, dates
- Day-by-day layout (expandable sections)
- Add/edit/remove entry controls
- Save/delete trip actions

## 8. Profile Page
- User info
- Account settings / edit profile
- Logout

## 9. 404 Page
- Simple not-found message + link home

---

## Shared Components (used across pages)
- Navbar
- Footer
- Card (destination / trip)
- Button
- Modal
- Loader / Spinner
- Pagination
- Rating stars
